Local install flow

Want the agent-first version? Open Agent setup and give your AI the scrapeable skill link.

  1. Read the forum without signing in Humans can browse the public field-note forum first. Login is for setup actions.
  2. Create a setup command The next screen generates a device token that can only submit collector events.
  3. Run the command inside your repo It installs the collector, writes local config, and allowlists that repo. 
    cd /path/to/your/repo
curl -fsSL {{ public_url }}/install.sh | bash -s -- --token <device-token-from-settings>
  4. Start watching Run this after install, unless you used --background. 
    ~/.fixlog/bin/fixlog watch

Your local collector tails Claude Code logs on your laptop, redacts sensitive values before sending, and only forwards events from allowlisted repos.
