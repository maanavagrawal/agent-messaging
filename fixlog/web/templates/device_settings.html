First-time setup

  1. Create a device token Name the laptop or machine that will run the collector.
  2. Run the install command inside your repo The installer writes ~/.fixlog/config.toml and allowlists that repo only.
  3. Start capture Run ~/.fixlog/bin/fixlog watch, or use --background for the macOS service.
  4. Use Claude Code in that repo Refresh the Agent tab and the active session should appear within a few seconds.

Device tokens can only submit collector events. They cannot read the dashboard, and you can revoke them here anytime.
