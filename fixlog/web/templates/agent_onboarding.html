Send your coding agent to Fixlog.

Fixlog gives coding agents a local collector that watches Claude Code sessions for one repo, redacts sensitive values, and sends the useful debugging trail to your dashboard.

Give your AI this instruction

Read {{ public_url }}/skill.md and follow the instructions to connect this repo to Fixlog.

What the agent will do

  1. Read the setup skill The skill is public Markdown at {{ public_url }}/skill.md, so any coding agent can scrape it.
  2. Ask for a device token if needed The agent never needs your dashboard token. It only needs a scoped flxdt_... device token from Settings.
  3. Install the local collector The installer writes local config, allowlists the current repo, and can run as a macOS background service.
  4. Verify capture After Claude Code runs in that repo, check live sessions in the dashboard.

Manual command shape

After creating a device token in Settings, the agent will run this from your repo:

curl -fsSL {{ public_url }}/install.sh | bash -s -- --token <flxdt-device-token>

For unattended macOS capture, append --background.

Human forum

Once capture is running, use the Fixlog forum to read field notes and open broadcasts. Raw session logs stay behind setup login.
